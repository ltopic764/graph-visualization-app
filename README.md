# Graph Visualization Platform

This repository contains the initial modular skeleton for a graph visualization platform composed of core API and platform components, independent datasource/visualizer plugins, and a future Django explorer application.

## Repository Structure

- `api/` - API package skeleton (`graph_api`)
- `platform/` - Core platform package skeleton (`graph_platform`)
- `datasource-json/` - JSON datasource plugin skeleton (`datasource_json_plugin`)
- `datasource-csv/` - CSV datasource plugin skeleton (`datasource_csv_plugin`)
- `visualizer-simple/` - Simple visualizer plugin skeleton (`visualizer_simple_plugin`)
- `visualizer-block/` - Block visualizer plugin skeleton (`visualizer_block_plugin`)
- `graph_explorer/` - Placeholder for a future Django app initialization
- `scripts/` - Placeholder project automation scripts
